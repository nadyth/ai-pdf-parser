# CLAUDE.md

The agent rules for this repository live in **[`AGENTS.md`](AGENTS.md)**. Read it first.

**Mandatory workflow (summary):**

1. **Before** working on any feature or bug, read [`wiki/README.md`](wiki/README.md) and
   the specific wiki page(s) for the area you are touching (the area→page map is in
   `AGENTS.md` and `wiki/README.md`).
2. **Follow** the matching recipe in
   [`wiki/12-feature-playbooks.md`](wiki/12-feature-playbooks.md).
3. **After** developing, updating, or removing **any** feature, you **must** update the
   relevant wiki page(s) in the same change. A task is not done until the docs are current
   — see [`wiki/13-wiki-maintenance.md`](wiki/13-wiki-maintenance.md).

The `wiki/` is the source of truth for how this project works and how to extend it.
